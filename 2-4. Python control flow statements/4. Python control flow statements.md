## `if` statement

`if` statement structure:

    1. 
``` python
if(expression):
    statement
```
    2. 
``` python
if(expression):
    statement
else:
    statement
```
    3. 
``` python
if(expression):
    statement
elif:
    statement
else:
    statement
```


```python
a=7
if a==7:
    print(a)
```

    7

## `for` statement

`for` statement structure:
    1.  
``` python
for i in <sequence>:
    statement
```
    2. 
``` python
for i in range(<range>):
    statement
```
    3.
``` python
for i in <some sequence or range>:
    statement
else:
    statement
```

## `continue` statement

The continue statement is used to skip the rest of **the code inside a loop for the current iteration only**. Loop does not terminate but continues on with the next iteration.

![](asset/continue-statement-flowchart.jpg)


```python
a=1
while a<7:
    a+=1
    print(a)
```

    2
    3
    4
    5
    6
    7



```python
a=1
while a<7:
    a+=1
    if a==3:
        continue
    print(a)
```

    2
    4
    5
    6
    7



```python
for i in range(1,7):
```
